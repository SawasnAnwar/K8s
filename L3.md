# Linux Commands Explanation

## 1. pwd

### Meaning
pwd = Print Working Directory

### Purpose
Shows your current location (directory) in Linux.

### Example

> [!code] Linux Command
> ```bash
> pwd
> ```

### Output

> [!success]
> ```bash
> /home/abdelrahman
> ```

### Explanation
You are currently inside:

- `/home` → users folder
- `abdelrahman` → your personal user directory

---

## 2. clear

### Purpose
Clears the terminal screen.

### Example

> [!code] Linux Command
> ```bash
> clear
> ```

### Shortcut

> [!tip]
> ```bash
> Ctrl + L
> ```

Both do the same thing.

---

## 3. sudo

### Meaning
sudo = Super User Do

### Purpose
Run commands with administrator (root) privileges.

### Example

> [!code] Linux Command
> ```bash
> sudo passwd root
> ```

### Explanation

- `passwd` → command to change password
- `root` → administrator account

This command changes the root password.

---

## 4. passwd

### Purpose
Change a user password.

### Example

> [!code] Linux Command
> ```bash
> passwd
> ```

Changes your own password.

Or:

> [!code] Linux Command
> ```bash
> sudo passwd root
> ```

Changes root password.

---

## 5. su

### Meaning
su = Switch User

### Purpose
Switch from current user to another user.

### Example

> [!code] Linux Command
> ```bash
> su - root
> ```

### Explanation

- `-` loads root environment
- `root` is administrator account

After entering password:

> [!success]
> ```bash
> root@abdelrahman-virtual-machine:~#
> ```

You are now root user.

---

## 6. whoami

### Meaning
Shows current logged-in username.

### Example

> [!code] Linux Command
> ```bash
> whoami
> ```

### Output

> [!success]
> ```bash
> abdelrahman
> ```

After switching to root:

> [!code] Linux Command
> ```bash
> whoami
> ```

Output:

> [!success]
> ```bash
> root
> ```

---

## 7. ls

### Meaning
ls = List

### Purpose
Show files and folders inside current directory.

### Example

> [!code] Linux Command
> ```bash
> ls
> ```

### Example Output

> [!success]
> ```bash
> Documents Downloads Music Pictures
> ```

---

# Important Symbols

## $

Example:

> [!code]
> ```bash
> abdelrahman@machine:~$
> ```

Means:

- normal user

---

## #

Example:

> [!code]
> ```bash
> root@machine:~#
> ```

Means:

- root/admin user

---

# Root Directory

## /

The highest directory in Linux filesystem.

Example:

> [!code]
> ```bash
> /
> ```

Everything starts from `/`

---

# Home Directory

## /home/username

Stores user personal files.

Example:

> [!code]
> ```bash
> /home/abdelrahman
> ```

---

# Root User Directory

## /root

Special home directory for administrator.

Example:

> [!code]
> ```bash
> /root
> ```
